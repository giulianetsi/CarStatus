Rotas da API:

URL padrão:
-localhost:8080/api/

Cliente:
Salvar:
-cliente/ (POST)
Buscar por ID:
-cliente/{id} (GET)
Buscar todos:
-cliente/ (GET)
Editar:
-cliente/{id} (PUT)
Deletar:
-cliente/{id} (DELETE)

Mecanico:
Salvar:
-mecanico/ (POST)
Buscar por ID:
-mecanico/{id} (GET)
Buscar todos:
-mecanico/ (GET)
Editar:
-mecanico/{id} (PUT)
Deletar:
-mecanico/{id} (DELETE)

Carro:
Salvar:
-carro/ (POST)
Buscar por ID do carro:
-carro/{id} (GET)
Buscar todos:
-carro/ (GET)
Editar:
-carro/{id} (Put)
Deletar:
-carro/{id} (DELETE)
Buscar todos por ID do dono:
-carro/dono/{id}

Manutenção:
Salvar:
-manutencao/ (POST)
Buscar por ID:
-manutencao/{id} (GET)
Buscar todos:
-manutencao/ (GET)
Editar:
-manutencao/{id} (DELETE)
Buscar por Id do cliente:
-manutencao/cliente/{id}
Buscar por Id do mecanico:
-manutencao/mecanico/{id}
Iniciar:
-manutencao/{id}/iniciar (POST)
Finalizar:
-manutencao/{id}/finalizar (POST)
Cancelar:
-manutencao/{id}/cancelar (POST)
-manutencao/{id}/cancelar
